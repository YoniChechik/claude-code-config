# Claude Code Configuration

This repository contains custom Claude Code configuration including:

- **Agents**: Custom agent configurations for code quality and review
- **Commands**: Slash commands for common workflows
- **Hooks**: Shell hooks for automated tasks
- **Coding Style**: Project-specific coding standards

## Installation

Install as an editable package:

```bash
uv add --editable git+https://github.com/YoniChechik/claude-code-config.git
```

Or install locally:

```bash
uv add --editable .claude/
```
